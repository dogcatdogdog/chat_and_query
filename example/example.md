2.1.2 核心功能逻辑 
功能清单:
自然语言问答(设备状态/任务信息/事件查询)
会话历史管理
快捷问题模板
上下文理解(多轮对话)
数据可视化建议
问题意图识别
知识库集成
数据模型核心参数:
AI对话会话 {
conversationId: 会话唯一标识
userId: 用户ID
title: 会话标题 (根据首个问题自动生成)
createdTime: 创建时间
lastMessageTime: 最后消息时间
// 消息列表
messages: [{
messageId: 消息ID
role: 角色 (user-用户/assistant-AI助手)
content: 消息内容
timestamp: 时间戳
// AI响应附加信息
intentType: 意图类型 (query-查询/analysis-分析/suggestion-建议)
dataSource: 数据来源 [{
moduleName: 模块名称 (如: 任务管理/设备管理)
apiCalled: 调用的API接口
dataReturned: 返回的数据摘要
}]
// 可视化数据【后面的迭代完成】
chartData: {
type: 图表类型 (bar/line/pie/table)
config: 图表配置
data: 图表数据
}
// 相关资源
relatedResources: [{
type: 资源类型 (device/task/event/report)
id: 资源ID
name: 资源名称
url: 跳转链接
}]
}]
// 会话元数据
messageCount: 消息总数
status: 会话状态 (active-活跃/archived-已归档)
}

交互流程:
1. 新建对话:
用户进入AI问答页面
↓
显示欢迎界面 + 快捷问题按钮
↓
用户点击快捷问题 "今日任务数"
↓
自动填充到输入框并发送
↓
创建新的会话:- 生成conversationId- 会话标题: "今日任务数"
↓
添加到左侧会话历史列表
2. 发送问题:
用户在输入框输入: "今天有多少个巡检任务?"
↓
点击发送按钮 (或按Enter键)
↓
前端添加用户消息气泡到界面
↓
显示AI思考状态:- 添加AI消息气泡- 显示"正在思考中..." + 脉冲动画
↓
调用后端API:- POST /api/ai/chat- 参数: {
conversationId: "xxx",
message: "今天有多少个巡检任务?",
context: [前3轮对话历史]
}
↓
后端处理流程:
1. DeepSeek理解问题意图 → 查询任务数量
2. 识别需要调用的API → GET /api/missions?date=today
3. 调用任务管理模块API获取数据
4. 数据返回: {total: 42, completed: 15, running: 8, pending: 19}
5. DeepSeek生成回答文本
↓
前端接收响应并渲染AI消息:- 移除"思考中"提示- 显示AI回答:
"根据系统数据,今天共有 **42个巡检任务**,其中:
已完成:15个
进行中:8个
待执行:19个
相比昨天增长了12%,主要集中在上午10:00-12:00时段。"- 回答包含:- Markdown格式化文本- 数值高亮显示- 相关资源链接 (可选)
3. 多轮对话:
用户继续追问: "哪些设备在线?"
↓
系统保留上下文,理解这是连续对话
↓
调用后端API:- 携带conversationId- 携带前几轮对话历史作为context
↓
DeepSeek分析:- 理解用户在询问设备状态- 调用设备管理API- 获取在线设备列表
↓
AI回答:
"当前系统中 **18台设备在线** (75% 在线率):
📊 设备分布:
• 在线: DJI-001, DJI-002, DJI-005, DJI-008... (共18台)
• 离线: 4台
• 任务中: 2台
建议检查离线设备 DJI-003, DJI-007 的连接状态。"
4. 复杂查询与分析:
用户问: "分析一下最近的AI告警事件"
↓
DeepSeek识别为分析类意图
↓
调用AI事件模块API:- GET /api/events?timeRange=7days- 返回最近7天的事件数据
↓
DeepSeek进行数据分析:- 统计事件类型分布- 识别高发时段- 提取异常模式
↓
AI回答(包含文本+可视化):
"最近7天共产生 **156条AI告警**,分析如下:
事件类型分布:
• 人员入侵: 68次 (43.6%)
• 车辆检测: 45次 (28.8%)
• 设备缺陷: 28次 (17.9%)
• 烟火识别: 15次 (9.6%)
异常发现:
高发时段: 14:00-16:00 (占32%)
• 周三告警量激增至35次,建议检查当日巡检路线
• 人员入侵事件集中在禁区A,建议加强该区域监控
[查看详细报告] [导出数据]"
↓
同时渲染可视化图表:【后续实现】
- 饼图: 事件类型分布- 柱状图: 每日告警趋势
5. 快捷操作:
AI回答中包含操作按钮:
"设备DJI-003已离线超过2小时,建议创建维护工单。"
[创建工单] 按钮

用户点击按钮
↓
